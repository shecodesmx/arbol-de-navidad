# 🎄 Árbol de Navidad

Animación de un árbol de Navidad creada con **HTML, CSS y JavaScript**.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript

---

## ✨ Características

- 🎄 Árbol de Navidad animado
- 💡 Efectos visuales
- 🌈 Diseño navideño
- ⚡ Ligero y rápido
- 📱 Compatible con navegadores modernos

---

## 📂 Estructura del proyecto

```
arbol-de-navidad/
│
├── arbol_navidad.html
├── style.css
├── script.js
```

---

## ▶️ Cómo usar

1. Descarga el proyecto
2. Abre `arbol_navidad.html` en tu navegador
